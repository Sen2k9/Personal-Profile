Leveraging Machine Learning Approach to Setup Software-Defined Network(SDN) Controller Rules During DDoS Attack

In this work, a machine learning based DDoS detection system has been implemented in a virtual SDN environment testbed. This system identifies whether an incoming traffic is a DDoS attack or normal. From the result, we can see that the model has 93% accuracy to detect DDoS attack with low false-positive rate. The clear advantage to identify each attack type is that SDN controller will only block those specific attack types by automatically defining rules in its controller which will add an extra security over the firewall rules. Though in this project I showed only predefined setup rules in controller during DDoS attack, in future this work will be extended for automatic rules set up in the controller. Besides, I will reduce the controller bottleneck to only detect DDoS attack and have the plan to implement other network attacks such as spoofing, U2R, R2L etc. To handle with more feature instead of nine, deep learning method will be used for feature extraction addition to a strong classifier.

  • Programming Language: Python
  • Network Simulator: MNet

Recommendation System for Netﬂix Prize Dataset Using Collaborative Filtering Algorithm

Project Description

Here I implemented the collaborative filtering algorithm and applied it to a dataset containing movie ratings.

  • User-user collaborative filtering algorithm has been implemented for making the predictions. I choose Java as a programming language.
  • The dataset which has been used is a small subset of the movie ratings data from the Netflix Prize.
  • The paper “Empirical Analysis of Predictive Algorithms for Collaborative Filtering” https://arxiv.org/ftp/arxiv/papers/1301/1301.7363.pdf is a good reference as well to understand the concept.
  • Predicted Score: The project is interested in whether the system would recommend a movie (that the user has not already reviewed) to a user or not. Specifically, the input given is a user-id, movie-id, and the system need to compute the predicted rating for that user-id w.r.t the input movie-id and print out the predicted rating.
  • Recommendation: Given a user-id and a year-of-release, the system recommends a movie for that user for that year-of-release.
  • Algorithm used: User-User Collaborative Filtering Approach
  • Programming Language: Java
